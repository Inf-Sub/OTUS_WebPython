Пару слов про Vue

  • The progressive JavaScript framework
  • Большую популярность получила вторая версия - Vue2
  • Tools: VSC extension
  • Tools: Vue.js devtools Chrome

Ещё пару слов про Vue

  • Использует JSX для компонентов
  • По синтаксису напоминает handlebars
  • По природе напоминает Angular (даже AngularJS, а не Angular *)
  • Честно - понравится, если Вам зайдёт
  • Достаточно многим зашло

Ещё пару слов про Vue

  • Это библиотека
  • Не фреймворк (такой как Angular)
  • Отвечает только за часть View (собственно, как React)
  • Ничего лишнего
  • Ну и lightweight, как с точки зрения кода, так и веса
